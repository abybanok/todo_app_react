# Список задач — React SPA

Учебное React-приложение с загрузкой задач из API и маршрутизацией.

🔗 **Онлайн-версия:** https://abybanok.github.io/todo_app_react/

## Функционал
- Главная страница со списком задач
- Страница About с описанием
- Маршрутизация без перезагрузки
- Загрузка данных из JSONPlaceholder API
- Обработка состояний загрузки и ошибок

## Технологии
- React
- React Router
- Vite
- JSONPlaceholder API
